MyTube

Tech Stack: HTML5, CSS3, JavaScript

  • Video Categories: 3 categories with 6 videos each (18 total video pages)
  • Navigation Bar: Fully functional navbar linking to Home, Categories, and Upload page (UI only)
  • Video Pages: Individual pages for each video with embedded iframe players
  • Page Structure: Total of 23 static pages built with clean, semantic HTML

GitHub Repo

What I learned: Gained hands-on experience with iframe embedding, multi-page site architecture, and maintaining consistent layout and navigation across static pages.

Level-Up Life

Tech Stack: HTML5, CSS3, JavaScript

  • Progression System: Level, EXP, streak tracking with dynamic EXP thresholds and milestone rewards
  • Daily Quests & Achievements: Randomized daily quests with completion checks and unlockable badges that grant EXP bonuses
  • Customizable Chores: Add, delete, and complete chores to earn EXP, with UI buttons generated dynamically
  • Prestige Mechanics: Reset level and EXP at level 100 for prestige points, retaining streaks, badges, and stats to encourage replayability

GitHub Repo

What I learned:Built a persistent client-side progression system with dynamic UI and modular quest and prestige logic using vanilla JavaScript.
